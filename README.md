# PrepPilot AI 🚀

PrepPilot AI is an AI-powered communication and mock interview platform built to help students improve speaking confidence, professional communication, and interview preparation.

## ✨ Features
- AI Mock Interviews
- Communication Improvement
- Daily Speaking Practice
- Group Discussion Topics
- AI Feedback & Scores

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- Supabase
- OpenAI API
- Lovable

## 🎯 Goal
The goal of this project is to help students become more confident and interview-ready using AI-powered practice and feedback.

## 🚀 Future Improvements
- Voice-based interviews
- Speech analysis
- Filler word detection
- Real-time AI feedback
